## About the Project
Neon Rebellion is a side-scroll 2D futuristic hack and slash game set in a dystopian megacity.  
Player takes a role of an experimental cyber-enhanced fighter breaking free from corporate control.

The game is focused on:
- High-speed melee combat
- Combo-based attacks
- Mobility and dodging
- Futuristic neon-punk aesthetics
- Progression and upgrades
